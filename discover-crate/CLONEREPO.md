#SKILL: clone espforge repo 
To complete this skill, run `~/.config/opencode/skills/espforge-devices/scripts/clonerepo.sh <device>` where <device> 
is the device user wants to add to espforge.

This script clone repo and creates a feature branch called feature/<device>
